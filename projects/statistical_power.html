Noah's Ark Survival Task

  • Technical Skills: JavaScript, HTML, CSS
  • Github URL: Project Link

Many experiments have been criticised for being underpowered. Journals are now asking authors to submit a sample size justification, based on statistical power, in their reports. For students with limited research experience, the prospect of managing multiple high-powered experiments can be daunting. I created a novel task which seeked to find how well students reasoned with the concept of statistical power. Participants were assigned to either a low-powered, medium-powered, or high-powered simulation of the task, and asked to complete two challenges using the same resource.
